Objectif du projet : 
- Apprendre a créer une application php avec connexion bd.
- La faire évoluer étape par étape et apprendre avec l'IA.
- Environnement : XAMPP et Base données MySQL


Fonctionnalités actuelles : 
- Gestion d'une page register + login
- Gestion d'une page de profil (profile.php)
- Page d'index.
- Sécurité :
-- Si la personne est pas loggué et essaye d'accéder à une page, redirection simple vers login.

BASE DE DONNÉES

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nom VARCHAR(100) NOT NULL,
    email VARCHAR(150) NOT NULL UNIQUE,
    mot_de_passe VARCHAR(255) NOT NULL,
    date_creation TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
